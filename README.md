# Anima-DiscordBot

## 1.Create config in your project
Create file `config.json` in folder `Data`
In config should be:
```json
{
  "token": "YOUR_TOKEN",
  "clientId": "YOUR_BOT_ID",
  "apiKey": "YOUR_GPT3_API_KEY",
}
```
  If you have free gpt api key, <br>
  if not, you can remove this line and event 'gptEvent'
